# 🎨 Random Background Color Changer

A simple, modern HTML/CSS/JS project where you can start and stop changing the background color randomly using two buttons.  
The colors are generated randomly at a fixed interval using `setInterval()` and stopped using `clearInterval()`.

---

## 🚀 Features
- **Start / Stop Buttons** — Start changes the background color every 500ms, Stop halts the process.
- **Random Colors** — Background color changes to a random RGB value.
- **Interval Control** — Prevents starting multiple intervals by checking if one is already running.
- **Modern UI** — Glassmorphism design with smooth hover effects.
- **Smooth Transitions** — Background changes with a fade effect.
---

## 🛠️ Technologies Used
- **HTML5**
- **CSS3** (Glassmorphism + Transitions)
- **JavaScript (ES6)** — `setInterval()` & `clearInterval()` for color changes
